# Input
A simple Input to allow users to carry out an action. This is a controlled component so a `value` and an `onChange` prop must be passed.

## Base
For use as a main call to action.

<Preview>
  <Story name='Primary'>
    {() => {
      const [value, setValue] = useState('')
      return (
        <Input name={'name'} value={value} onChange={setValue} />
      )
    }}
  </Story>
</Preview>
