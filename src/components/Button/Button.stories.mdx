# Button
A simple buttons to allow users to carry out an action.
 
## Primary
For use as a main call to action.

<Preview>
  <Story name='Primary'>
    <Button type={'Primary'}>Primary Button</Button>
  </Story>
</Preview>

## Secondary
Secondary buttons should be used sparingly, in instances where a primary button would have too much emphasis.

<Preview>
  <Story name='Secondary'>
    <Button type={'Secondary'}>Secondary Button</Button>
  </Story>
</Preview>

## Tertiary
Tertiary buttons should be used in conjunction with primary buttons where there are two actions.

<Preview>
  <Story name='Tertiary'>
    <Button type={'Tertiary'}>Tertiary Button</Button>
  </Story>
</Preview>

## Action
Action buttons should be used to indicate to the user that they can perform a data related action such as add, edit, delete etc, or can be used in places where there is limited space.

  <Story name='Action'>
