# Radio
A simple Radio to allow users to carry out an action. This is a controlled component so a `value` and an `onChange` prop must be passed.

## Base
For use to indicate a boolean value.

<Preview>
  <Story name='Primary'>
    {() => {
      const [value, setValue] = useState(false)
      return (
        <Radio value={value} onChange={setValue} />
      )
    }}
  </Story>
</Preview>

## Group
For use to indicate multiple boolean values.

<Preview>
  <Story name='Group'>
    {() => {
      const [value, setValue] = useState(null)
      return (
        <div>
          <p className='p'>Value selected: {JSON.stringify(value)}</p>
          <Radio name={'favourite-animal'} value={value === 'cat'} onChange={(value) => setValue('cat', value)} />
          <Radio name={'favourite-animal'} value={value === 'dog'} onChange={(value) => setValue('dog', value)} />
          <Radio name={'favourite-animal'} value={value === 'horse'} onChange={(value) => setValue('horse', value)} />
        </div>
      )
    }}
  </Story>
</Preview>
