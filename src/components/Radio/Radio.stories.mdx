# Radio
A simple Radio to allow users to carry out an action. This is a controlled component so a `value` and an `onChange` prop must be passed.

## Base
For use to indicate a boolean value.

<Preview>
  <Story name='Base'>
    {() => {
      const [value, setValue] = useState(false)
      return (
        <div>
          <Text>Value: {`${value}`}</Text>
          <Radio id={'radio'} value={value} onChange={setValue}>Radio</Radio>
          <Radio id={'radio-disabled'} disabled uncontrolled>Disabled</Radio>
          <Radio id={'radio-disabled-checked'} value={true} disabled uncontrolled>Disabled</Radio>
        </div> 
      )
    }}
  </Story>
</Preview>

## Group
For use to indicate multiple boolean values.

<Preview>
  <Story name='Group'>
    {() => {
      const [value, setValue] = useState(null)
      return (
        <div>
          <Text>Value selected: {JSON.stringify(value)}</Text>
          <Radio name={'radio'} value={value === 'cat'} onChange={(value) => setValue('cat', value)}>Cat</Radio>
          <Radio name={'radio'} value={value === 'dog'} onChange={(value) => setValue('dog', value)}>Dog</Radio>
          <Radio name={'radio'} value={value === 'horse'} onChange={(value) => setValue('horse', value)}>Horse</Radio>
        </div>
      )
    }}
  </Story>
</Preview>
